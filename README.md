# Protótipo de Combate por Turno
Projeto com o objetivo de criar um protótipo de sistema de combate por turno. O projeto foi desenvolvido em C# utilizando Windows Forms, permitindo criar uma interface gráfica funcional para simular o combate de forma visual e interativa.

## 🎮 Funcionalidades
- Sistema de combate por turno
- Alternância entre jogador e inimigo
- Sistema de ataque com cálculo de dano
- Controle de pontos de vida (HP)
- Verificação de condição de vitória ou derrota
- Interface gráfica interativa

# Como Executar
1️⃣ Abra o projeto no Visual Studio

2️⃣ Compile a solução

3️⃣ Execute o projeto (F5 ou botão "Start")

A interface do combate será exibida em uma janela Windows Forms.
